# config-show
Show pypyr's current configuration settings and sources. This is useful to
troubleshoot your configuration settings.

Run me like this:
```text
$ pypyr config-show
```

 See
[troubleshooting config]({{< ref "/docs/getting-started/config#troubleshooting">}})
for details.
